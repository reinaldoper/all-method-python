# Repositório Python

## Este repositório contém um conjunto de arquivos Python que demonstram conceitos fundamentais da linguagem, incluindo lista, dicionários, programação orientada a objetos (POO), interface, herança, polimorfismo, classes, encapsulamento, banco de dados, iteradores, geradores, docker e abstração.

## Estrutura do Repositório
- lista.py: Implementação de operações básicas em listas.
- dicionarios.py: Exemplos de manipulação de dicionários em Python.
- poo.py: Demonstração de conceitos de POO, incluindo definição de classes e métodos.
- interface.py: Exemplo de uma interface básica utilizando classes em Python.
- heranca.py: Implementação de classes e subclasse para demonstrar herança.
- polimorfismo.py: Exemplos de polimorfismo utilizando métodos sobrescritos.
- abstract.py: Demonstração de abstração através de classes abstratas em Python.

## Utilização
- Cada arquivo Python pode ser executado individualmente para observar o comportamento dos conceitos demonstrados. 
- Para executar um arquivo, utilize o comando python nome_do_arquivo.py.
